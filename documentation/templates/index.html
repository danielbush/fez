Project Overview

Mission and Scope

What problem does this project address?
The project addresses the lack of a suitable freely available and open-source web based Digital Repository, specifically for use in Libraries.
What is the goal of this project?

To provide a sustainable web based digital repository for research documents and data at the University of Queensland Library.

eSpace is an open source project to produce and maintain a highly flexible web interface to FEDORA for any Library or Insituition to configure and publish or archive documents of any type sustainably.

What is the scope of this project?
This project is expected to last the entire APSR & eScholarship testbed project lifetime of 2005-2006.

Status

We have completed some basic functionality described in our product specification and are working on finishing off the authentication and authorisation model to release version 0.1.0.

The next major milestone is the 0.2.0 release with workflow functionality and a wider set of testers.
