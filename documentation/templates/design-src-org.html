Overview

What are the most important facts that a developer should know about this source code organization and build system?
The system configuration settings (application and DB usernames and passwords) are stored in config.inc.php. The Fez code architecture uses the Smarty templating system to seperate business logic from presentation logic. Some PEAR modules are used for some core functions.
What are the ranked goals of this source code organization and build system?
  1. Separation of files by type (classes, smarty templates, javascript, php runtime scripts)
  2. Platform independence
