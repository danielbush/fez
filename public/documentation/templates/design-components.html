Release Information

Project: Fez
Internal Release Number: 1.2.0 Beta

C-00: Fez PHP Web Application

Description: The Web Application Frontend to the FEDORA digital repository
Environmental Constraints: A Web Server running PHP 5
Available Interfaces: HTML web interface for end users. OAI-PMH interface for harversters.

C-10: Fedora Digital Repository

Description: http://www.fedora.info/
Environmental Constraints: A server capable of running the Fedora inbuilt Tomcat server.
Available Interfaces: Exposed FEDORA API web services for access and management. Fedora Administrator client GUI.

C-20: MySQL Database

Description: A database for the Fez specific workflow, security and document management. Also required for the Fedora structural control of digital objects.
Environmental Constraints: A server capable of running MySQL 4.1+
Available Interfaces: MySQL interface to the PHP code. MySQL administrator access for admin tasks.

C-21: Storage Area Network

Description: A file storage mapped network drive to a large amount of space.
Environmental Constraints: A network location with enough storage (Terabytes) to store all the datastreams of digital objects in Fez, eg PDFs, Images, Audio etc
Available Interfaces: Traditional file storage interfaces, restricted by access control.
